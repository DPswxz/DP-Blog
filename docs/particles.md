# 日语常用助词

助词不仅标明了句子成分（如主语、宾语），还表达了情感和语境。

## 主要助词列表

### 1. 话题与主语

- **は** (读作わ): 主题标记
  - <ruby>私<rt>わたし</rt></ruby>は<ruby>学生<rt>がくせい</rt></ruby>です。
    - 我是学生。
  - <ruby>今日<rt>きょう</rt></ruby>は<ruby>暑<rt>あつ</rt></ruby>いです。
    - 今天很热。
  - *は* 之前的内容是谈论的那个主题。

- **が**: 主语/焦点标记
  - <ruby>雨<rt>あめ</rt></ruby>が<ruby>降<rt>ふ</rt></ruby>っています。
    - 下雨了。
  - <ruby>私<rt>わたし</rt></ruby>が<ruby>行<rt>い</rt></ruby>きます。
    - *我*去 (强调是我，不是别人)。
  - <ruby>犬<rt>いぬ</rt></ruby>が<ruby>好<rt>す</rt></ruby>きです。
    - 喜欢狗 (对象语)。

### 2. 对象与场所

- **を** (读作お): 直接宾语标记 / 经过的场所
  - <ruby>本<rt>ほん</rt></ruby>を<ruby>読<rt>よ</rt></ruby>みます。
    - 读书。
  - <ruby>公園<rt>こうえん</rt></ruby>を<ruby>散歩<rt>さんぽ</rt></ruby>します。
    - 在公园散步 (移动经过)。

- **に**: 时间、地点、目的地、对象
  - <ruby>学校<rt>がっこう</rt></ruby>に<ruby>行<rt>い</rt></ruby>きます。
    - 去学校 (目的地)。
  - <ruby>8時<rt>はちじ</rt></ruby>に<ruby>起<rt>お</rt></ruby>きます。
    - 8点起床 (时间点)。
  - <ruby>彼<rt>かれ</rt></ruby>に<ruby>会<rt>あ</rt></ruby>います。
    - 见他 (对象)。
  - <ruby>部屋<rt>へや</rt></ruby>にいます。
    - 在房间里 (存在地点)。

- **で**: 动作发生的场所、手段/工具
  - <ruby>図書館<rt>としょかん</rt></ruby>で<ruby>勉強<rt>べんきょう</rt></ruby>します。
    - 在图书馆学习 (动作地点)。
  - バスで<ruby>行<rt>い</rt></ruby>きます。
    - 坐巴士去 (手段)。
  - <ruby>箸<rt>はし</rt></ruby>で<ruby>食<rt>た</rt></ruby>べます。
    - 用筷子吃 (工具)。

- **へ** (读作え): 方向/目的地 (比 `に` 更强调方向)
  - <ruby>日本<rt>にほん</rt></ruby>へ<ruby>行<rt>い</rt></ruby>きます。
    - 去日本。
  - <ruby>家<rt>うち</rt></ruby>へ<ruby>帰<rt>かえ</rt></ruby>ります。
    - 回家。

### 3. 其他常用助词

- **も**: 也，强调
  - <ruby>私<rt>わたし</rt></ruby>も<ruby>学生<rt>がくせい</rt></ruby>です。
    - 我也是学生。
  - これも<ruby>下<rt>くだ</rt></ruby>さい。
    - 请也给我这个。

- **の**: 属性、所有格
  - <ruby>私<rt>わたし</rt></ruby>の<ruby>本<rt>ほん</rt></ruby>。
    - 我的书。
  - <ruby>日本語<rt>にほんご</rt></ruby>の<ruby>先生<rt>せんせい</rt></ruby>。
    - 日语老师。

- **と**: 和，引用
  - <ruby>貴方<rt>あなた</rt></ruby>と<ruby>私<rt>わたし</rt></ruby>。
    - 你和我。
  - 「こんにちは」と<ruby>言<rt>い</rt></ruby>いました。
    - 说了“你好”。
  - <ruby>友達<rt>ともだち</rt></ruby>と<ruby>遊<rt>あそ</rt></ruby>びます。
    - 和朋友玩。

- **か**: 疑问，或者
  - これかそれ。
    - 这个或者那个。
  - <ruby>行<rt>い</rt></ruby>きますか？
    - 去吗？

- **から**: 从... (起点/原因)
  - <ruby>東京<rt>とうきょう</rt></ruby>から<ruby>来<rt>き</rt></ruby>ました。
    - 来自东京。
  - <ruby>朝<rt>あさ</rt></ruby>から<ruby>晩<rt>ばん</rt></ruby>まで。
    - 从早到晚。
  - <ruby>寒<rt>さむ</rt></ruby>いから<ruby>行<rt>い</rt></ruby>きません。
    - 因为冷所以不去。

- **まで**: 到... (终点)
  - <ruby>駅<rt>えき</rt></ruby>まで<ruby>走<rt>はし</rt></ruby>ります。
    - 跑到车站。
  - 5<ruby>時<rt>じ</rt></ruby>まで<ruby>働<rt>はたら</rt></ruby>きます。
    - 工作到5点。

- **や**: ...和...等 (列举，暗示还有其他)
  - <ruby>本<rt>ほん</rt></ruby>やペンを買いました。
    - 买了书、笔等等。

- **ね**: ...吧/呢 (寻求认同)
  - <ruby>暑<rt>あつ</rt></ruby>いですね。
    - 好热啊。

- **よ**: ...哟/哦 (告知信息)
  - <ruby>明日<rt>あした</rt></ruby>は<ruby>試験<rt>しけん</rt></ruby>ですよ。
    - 明天考试哦。
