# 形容词详解

日语形容词分为 **い<ruby>形容詞<rt>けいようし</rt></ruby>** 和 **な<ruby>形容詞<rt>けいようし</rt></ruby>** 两大类。

## い<ruby>形容詞<rt>けいようし</rt></ruby>

以「い」结尾。可以直接修饰名词，也可以做谓语。

### 常见い形容词

| 日语 | 意思 | 日语 | 意思 |
|---|---|---|---|
| <ruby>大<rt>おお</rt></ruby>きい | 大 | <ruby>小<rt>ちい</rt></ruby>さい | 小 |
| <ruby>高<rt>たか</rt></ruby>い | 高/贵 | <ruby>安<rt>やす</rt></ruby>い | 便宜 |
| <ruby>新<rt>あたら</rt></ruby>しい | 新 | <ruby>古<rt>ふる</rt></ruby>い | 旧 |
| <ruby>暑<rt>あつ</rt></ruby>い | 热 | <ruby>寒<rt>さむ</rt></ruby>い | 冷 |
| <ruby>美味<rt>おい</rt></ruby>しい | 好吃 | まずい | 难吃 |
| <ruby>楽<rt>たの</rt></ruby>しい | 快乐 | <ruby>悲<rt>かな</rt></ruby>しい | 悲伤 |
| <ruby>忙<rt>いそが</rt></ruby>しい | 忙 | <ruby>難<rt>むずか</rt></ruby>しい | 难 |
| <ruby>易<rt>やさ</rt></ruby>しい | 简单/温柔 | <ruby>面白<rt>おもしろ</rt></ruby>い | 有趣 |

### い形容词活用表

以「<ruby>高<rt>たか</rt></ruby>い」为例：

| 活用 | 形式 | 例句 |
|---|---|---|
| 谓语 (非过去) | <ruby>高<rt>たか</rt></ruby>い | この<ruby>本<rt>ほん</rt></ruby>は<ruby>高<rt>たか</rt></ruby>い。（这本书贵。） |
| 谓语 (过去) | <ruby>高<rt>たか</rt></ruby>**かった** | <ruby>去年<rt>きょねん</rt></ruby>は<ruby>高<rt>たか</rt></ruby>かった。（去年很贵。） |
| 否定 (非过去) | <ruby>高<rt>たか</rt></ruby>**くない** | あまり<ruby>高<rt>たか</rt></ruby>くない。（不太贵。） |
| 否定 (过去) | <ruby>高<rt>たか</rt></ruby>**くなかった** | <ruby>高<rt>たか</rt></ruby>くなかった。（以前不贵。） |
| 修饰名词 | <ruby>高<rt>たか</rt></ruby>い + 名词 | <ruby>高<rt>たか</rt></ruby>い<ruby>建物<rt>たてもの</rt></ruby>。（高的建筑。） |
| て形 (连接) | <ruby>高<rt>たか</rt></ruby>**くて** | <ruby>高<rt>たか</rt></ruby>くて<ruby>美味<rt>おい</rt></ruby>しい。（贵而且好吃。） |

## な<ruby>形容詞<rt>けいようし</rt></ruby>

也叫「<ruby>形容動詞<rt>けいようどうし</rt></ruby>」。修饰名词时需要加「な」。

### 常见な形容词

| 日语 | 意思 | 日语 | 意思 |
|---|---|---|---|
| <ruby>綺麗<rt>きれい</rt></ruby> | 漂亮/干净 | <ruby>有名<rt>ゆうめい</rt></ruby> | 有名 |
| <ruby>静<rt>しず</rt></ruby>か | 安静 | にぎやか | 热闹 |
| <ruby>元気<rt>げんき</rt></ruby> | 精神/健康 | <ruby>好<rt>す</rt></ruby>き | 喜欢 |
| <ruby>嫌<rt>きら</rt></ruby>い | 讨厌 | <ruby>上手<rt>じょうず</rt></ruby> | 擅长 |
| <ruby>下手<rt>へた</rt></ruby> | 不擅长 | <ruby>大変<rt>たいへん</rt></ruby> | 辛苦/严重 |
| <ruby>大切<rt>たいせつ</rt></ruby> | 重要 | <ruby>便利<rt>べんり</rt></ruby> | 方便 |

### な形容词活用表

以「<ruby>静<rt>しず</rt></ruby>か」为例：

| 活用 | 形式 | 例句 |
|---|---|---|
| 谓语 (非过去) | <ruby>静<rt>しず</rt></ruby>かです | この<ruby>部屋<rt>へや</rt></ruby>は<ruby>静<rt>しず</rt></ruby>かです。（这个房间安静。） |
| 谓语 (过去) | <ruby>静<rt>しず</rt></ruby>か**でした** | <ruby>昨日<rt>きのう</rt></ruby>は<ruby>静<rt>しず</rt></ruby>かでした。（昨天很安静。） |
| 否定 (非过去) | <ruby>静<rt>しず</rt></ruby>か**ではありません** | <ruby>静<rt>しず</rt></ruby>かではありません。（不安静。） |
| 否定 (过去) | <ruby>静<rt>しず</rt></ruby>か**ではありませんでした** | <ruby>静<rt>しず</rt></ruby>かではありませんでした。（以前不安静。） |
| 修饰名词 | <ruby>静<rt>しず</rt></ruby>か**な** + 名词 | <ruby>静<rt>しず</rt></ruby>かな<ruby>場所<rt>ばしょ</rt></ruby>。（安静的地方。） |
| て形 (连接) | <ruby>静<rt>しず</rt></ruby>か**で** | <ruby>静<rt>しず</rt></ruby>かで<ruby>綺麗<rt>きれい</rt></ruby>です。（安静又漂亮。） |

## 特殊注意

以下词虽以「い」结尾，但属于 **な形容词**：

| 日语 | 意思 |
|---|---|
| <ruby>綺麗<rt>きれい</rt></ruby> | 漂亮/干净 |
| <ruby>嫌<rt>きら</rt></ruby>い | 讨厌 |
| <ruby>有名<rt>ゆうめい</rt></ruby> | 有名 |

这些词修饰名词时要加「な」，而不是直接接名词。

<ruby>綺麗<rt>きれい</rt></ruby>**な**<ruby>花<rt>はな</rt></ruby>。（漂亮的花。）✅
