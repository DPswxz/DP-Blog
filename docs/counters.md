# 数字与日期

日语有两套主要的数字读音系统，分别是**训读** (本土读法) 和**音读** (汉字读法)。根据计数对象的不同，使用的读音也不同。

## 基础数字 (1-10)

| 数字 | 音读 (用于大多数场合) | 训读 (用于通用计数) |
|---|---|---|
| 1 | <ruby>一<rt>いち</rt></ruby> | <ruby>一<rt>ひと</rt></ruby>つ |
| 2 | <ruby>二<rt>に</rt></ruby> | <ruby>二<rt>ふた</rt></ruby>つ |
| 3 | <ruby>三<rt>さん</rt></ruby> | <ruby>三<rt>みっ</rt></ruby>つ |
| 4 | <ruby>四<rt>よん</rt></ruby> / <ruby>し<rt></rt></ruby> | <ruby>四<rt>よっ</rt></ruby>つ |
| 5 | <ruby>五<rt>ご</rt></ruby> | <ruby>五<rt>いつ</rt></ruby>つ |
| 6 | <ruby>六<rt>ろく</rt></ruby> | <ruby>六<rt>むっ</rt></ruby>つ |
| 7 | <ruby>七<rt>なな</rt></ruby> / <ruby>しち<rt></rt></ruby> | <ruby>七<rt>なな</rt></ruby>つ |
| 8 | <ruby>八<rt>はち</rt></ruby> | <ruby>八<rt>やっ</rt></ruby>つ |
| 9 | <ruby>九<rt>きゅう</rt></ruby> / <ruby>く<rt></rt></ruby> | <ruby>九<rt>ここの</rt></ruby>つ |
| 10 | <ruby>十<rt>じゅう</rt></ruby> | <ruby>十<rt>とお</rt></ruby> |

## 日期 (1日-10日)

日期的读法中，1日到10日较为特殊，多用训读。

- **1日**: <ruby>一日<rt>ついたち</rt></ruby>
- **2日**: <ruby>二日<rt>ふつか</rt></ruby>
- **3日**: <ruby>三日<rt>みっか</rt></ruby>
- **4日**: <ruby>四日<rt>よっか</rt></ruby>
- **5日**: <ruby>五日<rt>いつか</rt></ruby>
- **6日**: <ruby>六日<rt>むいか</rt></ruby>
- **7日**: <ruby>七日<rt>なのか</rt></ruby>
- **8日**: <ruby>八日<rt>ようか</rt></ruby>
- **9日**: <ruby>九日<rt>ここのか</rt></ruby>
- **10日**: <ruby>十日<rt>とおか</rt></ruby>
- **20日**: <ruby>二十日<rt>はつか</rt></ruby> (例外)

其他日期通常是 **数字 + <ruby>日<rt>にち</rt></ruby>**。

## 常见量词

量词的使用取决于物品的形状或性质。

- **<ruby>個<rt>こ</rt></ruby>**: 最通用的量词，指圆形或立体的物体 (如苹果、鸡蛋)。
- **<ruby>本<rt>ほん</rt></ruby>**: 细长的物体 (如笔、瓶子、树)。
- **<ruby>枚<rt>まい</rt></ruby>**: 扁平的物体 (如纸、衬衫、照片)。
- **<ruby>冊<rt>さつ</rt></ruby>**: 书籍、杂志。
- **<ruby>台<rt>だい</rt></ruby>**: 机械、车辆、家电。
- **<ruby>人<rt>にん</rt></ruby>**: 人数 (1人：<ruby>一人<rt>ひとり</rt></ruby>，2人：<ruby>二人<rt>ふたり</rt></ruby>，3人及以上：数字+にん)。
- **<ruby>匹<rt>ひき</rt></ruby>**: 小动物 (如猫、狗)。
- **<ruby>頭<rt>とう</rt></ruby>**: 大动物 (如大象、鲸鱼)。
