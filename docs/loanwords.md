# 外来语

外来语主要用**片假名**书写，大多源自英语。掌握外来语可以快速扩充词汇量。

|---|---|---|
| コーヒー | coffee | 咖啡 |
| ジュース | juice | 果汁 |
| ビール | bier (德语) | 啤酒 |
| パン | pão (葡语) | 面包 |
| ケーキ | cake | 蛋糕 |
| サラダ | salad | 沙拉 |
| レストラン | restaurant | 餐厅 |

| アイスクリーム | ice cream | 冰淇淋 |

## 外来语的读音规律

日语中没有的音会被替换成最接近的发音：

| 原音 | 日语替换 | 示例 |
|---|---|---|
| **l** / **r** | ラ行 (ラリルレロ) | light → ライト |
| **v** | バ行 (バビブベボ) | violin → バイオリン |
| **th** | サ行 / ザ行 | three → スリー |
| **ti** | チ | ticket → チケット |
| **si** | シ | system → システム |
| 词尾辅音 | 加ウ段假名 | bed → ベッ**ド**、cup → カッ**プ** |
