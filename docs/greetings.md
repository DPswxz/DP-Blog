# 日常寒暄

掌握基本的问候语是日语交流的第一步。

## 见面问候

- **おはようございます**
  - 早上好 (敬语)。对朋友可以说「おはよう」。
- **こんにちは**
  - 你好 / 下午好。用于白天见面时。
- **こんばんは**
  - 晚上好。用于晚上见面时。
- **お<ruby>休<rt>やす</rt></ruby>みなさい**
  - 晚安。睡前或晚上分开时说。

## 告别

- **さようなら**
  - 再见。通常用于较长时间的分离。
- **じゃ、また**
  - 待会儿见 / 回头见。朋友之间常用。
- **<ruby>失礼<rt>しつれい</rt></ruby>します**
  - 告辞了 / 失礼了。用于挂电话或离开上级/长辈时。

## 感谢与道歉

- **ありがとうございます**
  - 谢谢。
- **すみません**
  - 对不起 / 不好意思 / 劳驾。用于道歉或引起注意。
- **ごめんなさい**
  - 对不起。主要用于道歉。

## 吃饭

- **<ruby>頂<rt>いただ</rt></ruby>きます**
  - 我开动了。饭前说。
- **ご<ruby>馳走様<rt>ちそうさま</rt></ruby>でした**
  - 多谢款待 / 我吃饱了。饭后说。

## 出门与回家

- **<ruby>行<rt>い</rt></ruby>ってきます**
  - 我出门了。
- **<ruby>行<rt>い</rt></ruby>ってらっしゃい**
  - 慢走 / 路上小心。对出门的人说。
- **ただいま**
  - 我回来了。
- **お<ruby>帰<rt>かえ</rt></ruby>りなさい**
  - 欢迎回来 / 你回来了。
