# 敬语入门

日语敬语体系分为三大类：**<ruby>尊敬語<rt>そんけいご</rt></ruby>**、**<ruby>謙譲語<rt>けんじょうご</rt></ruby>** 和 **<ruby>丁寧語<rt>ていねいご</rt></ruby>**。

## <ruby>丁寧語<rt>ていねいご</rt></ruby>（礼貌语）

最基础的敬语形式，日常会话中最常用。

| 普通形 | 丁寧形 | 意思 |
|---|---|---|
| <ruby>食<rt>た</rt></ruby>べる | <ruby>食<rt>た</rt></ruby>べます | 吃 |
| <ruby>行<rt>い</rt></ruby>く | <ruby>行<rt>い</rt></ruby>きます | 去 |
| <ruby>見<rt>み</rt></ruby>る | <ruby>見<rt>み</rt></ruby>ます | 看 |
| する | します | 做 |

## <ruby>尊敬語<rt>そんけいご</rt></ruby>（尊敬语）

用于抬高**对方**或**第三者**的动作。

### 常见替换动词

| 普通动词 | 尊敬语 | 意思 |
|---|---|---|
| <ruby>食<rt>た</rt></ruby>べる / <ruby>飲<rt>の</rt></ruby>む | <ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がる | 吃 / 喝 |
| <ruby>行<rt>い</rt></ruby>く / <ruby>来<rt>く</rt></ruby>る | いらっしゃる | 去 / 来 |
| いる | いらっしゃる | 在 |
| <ruby>言<rt>い</rt></ruby>う | おっしゃる | 说 |
| する | なさる | 做 |
| <ruby>見<rt>み</rt></ruby>る | ご<ruby>覧<rt>らん</rt></ruby>になる | 看 |
| <ruby>知<rt>し</rt></ruby>っている | ご<ruby>存<rt>ぞん</rt></ruby>じです | 知道 |

### お/ご ～ になる 句型

大部分动词可以用这个万能句型转换为尊敬语：

> **お** + 动词ます形去掉ます + **になる**

<ruby>先生<rt>せんせい</rt></ruby>がお<ruby>書<rt>か</rt></ruby>きになりました。
- 老师写了。

<ruby>社長<rt>しゃちょう</rt></ruby>がお<ruby>帰<rt>かえ</rt></ruby>りになりました。
- 社长回去了。

## <ruby>謙譲語<rt>けんじょうご</rt></ruby>（谦让语）

用于压低**自己**的动作来表示对对方的敬意。

### 常见替换动词

| 普通动词 | 谦让语 | 意思 |
|---|---|---|
| <ruby>食<rt>た</rt></ruby>べる / <ruby>飲<rt>の</rt></ruby>む | いただく | 吃 / 喝 |
| <ruby>行<rt>い</rt></ruby>く / <ruby>来<rt>く</rt></ruby>る | <ruby>参<rt>まい</rt></ruby>る | 去 / 来 |
| いる | おる | 在 |
| <ruby>言<rt>い</rt></ruby>う | <ruby>申<rt>もう</rt></ruby>す | 说 |
| する | いたす | 做 |
| <ruby>見<rt>み</rt></ruby>る | <ruby>拝見<rt>はいけん</rt></ruby>する | 看 |
| <ruby>知<rt>し</rt></ruby>っている | <ruby>存<rt>ぞん</rt></ruby>じております | 知道 |
| もらう | いただく | 得到 |
| あげる | <ruby>差<rt>さ</rt></ruby>し<ruby>上<rt>あ</rt></ruby>げる | 给 |

### お/ご ～ する 句型

> **お** + 动词ます形去掉ます + **する**

お<ruby>荷物<rt>にもつ</rt></ruby>をお<ruby>持<rt>も</rt></ruby>ちします。
- 我来帮您拿行李。

ご<ruby>案内<rt>あんない</rt></ruby>いたします。
- 由我来为您引路。

## 实际场景对照

| 场景 | 普通 | 尊敬语 (对方) | 谦让语 (自己) |
|---|---|---|---|
| 吃 | <ruby>食<rt>た</rt></ruby>べる | <ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がる | いただく |
| 去 | <ruby>行<rt>い</rt></ruby>く | いらっしゃる | <ruby>参<rt>まい</rt></ruby>る |
| 说 | <ruby>言<rt>い</rt></ruby>う | おっしゃる | <ruby>申<rt>もう</rt></ruby>す |
| 做 | する | なさる | いたす |
| 看 | <ruby>見<rt>み</rt></ruby>る | ご<ruby>覧<rt>らん</rt></ruby>になる | <ruby>拝見<rt>はいけん</rt></ruby>する |
