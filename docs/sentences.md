# 常用句型

掌握常用句型是灵活运用日语的关键。

## 1. ～たい（想做…）

表达自己的愿望。将动词ます形去掉ます + たい。

<ruby>日本<rt>にほん</rt></ruby>に<ruby>行<rt>い</rt></ruby>き**たい**です。
- 想去日本。

<ruby>寿司<rt>すし</rt></ruby>を<ruby>食<rt>た</rt></ruby>べ**たい**です。
- 想吃寿司。

<ruby>映画<rt>えいが</rt></ruby>を<ruby>見<rt>み</rt></ruby>**たい**です。
- 想看电影。

## 2. ～てください（请做…）

表示请求。将动词变为て形 + ください。

<ruby>窓<rt>まど</rt></ruby>を<ruby>開<rt>あ</rt></ruby>け**てください**。
- 请开窗。

もう<ruby>一度<rt>いちど</rt></ruby><ruby>言<rt>い</rt></ruby>っ**てください**。
- 请再说一遍。

ここに<ruby>名前<rt>なまえ</rt></ruby>を<ruby>書<rt>か</rt></ruby>い**てください**。
- 请在这里写名字。

## 3. ～ている（正在… / 状态）

表示进行时或持续状态。动词て形 + いる。

<ruby>今<rt>いま</rt></ruby>、<ruby>本<rt>ほん</rt></ruby>を<ruby>読<rt>よ</rt></ruby>ん**でいます**。
- 现在正在读书。

<ruby>東京<rt>とうきょう</rt></ruby>に<ruby>住<rt>す</rt></ruby>ん**でいます**。
- 住在东京。（持续状态）

<ruby>結婚<rt>けっこん</rt></ruby>し**ています**。
- 已经结婚了。（结果状态）

## 4. ～てもいい（可以做…）

表示许可。动词て形 + もいい。

<ruby>写真<rt>しゃしん</rt></ruby>を<ruby>撮<rt>と</rt></ruby>っ**てもいい**ですか？
- 可以拍照吗？

ここに<ruby>座<rt>すわ</rt></ruby>っ**てもいい**ですか？
- 可以坐这里吗？

## 5. ～てはいけない（不可以做…）

表示禁止。动词て形 + はいけない。

ここでタバコを<ruby>吸<rt>す</rt></ruby>っ**てはいけません**。
- 不可以在这里吸烟。

<ruby>授業中<rt>じゅぎょうちゅう</rt></ruby>にスマホを<ruby>使<rt>つか</rt></ruby>っ**てはいけません**。
- 上课时不可以用手机。

## 6. ～ことができる（能够做…）

表达能力。动词辞书形 + ことができる。

<ruby>日本語<rt>にほんご</rt></ruby>を<ruby>話<rt>はな</rt></ruby>す**ことができます**。
- 会说日语。

<ruby>車<rt>くるま</rt></ruby>を<ruby>運転<rt>うんてん</rt></ruby>する**ことができます**。
- 会开车。

## 7. ～たことがある（曾经做过…）

表达经历。动词た形 + ことがある。

<ruby>富士山<rt>ふじさん</rt></ruby>に<ruby>登<rt>のぼ</rt></ruby>っ**たことがあります**。
- 爬过富士山。

<ruby>日本料理<rt>にほんりょうり</rt></ruby>を<ruby>作<rt>つく</rt></ruby>っ**たことがあります**。
- 做过日本料理。

## 8. ～つもり（打算做…）

表达意图。动词辞书形 + つもりです。

<ruby>来年<rt>らいねん</rt></ruby>、<ruby>日本<rt>にほん</rt></ruby>に<ruby>行<rt>い</rt></ruby>く**つもりです**。
- 打算明年去日本。

<ruby>大学<rt>だいがく</rt></ruby>で<ruby>日本語<rt>にほんご</rt></ruby>を<ruby>勉強<rt>べんきょう</rt></ruby>する**つもりです**。
- 打算在大学学日语。

## 9. ～なければならない（必须做…）

表达义务。动词ない形去掉い + ければならない。

<ruby>宿題<rt>しゅくだい</rt></ruby>をし**なければなりません**。
- 必须做作业。

<ruby>毎日<rt>まいにち</rt></ruby><ruby>薬<rt>くすり</rt></ruby>を<ruby>飲<rt>の</rt></ruby>ま**なければなりません**。
- 必须每天吃药。

## 10. ～と<ruby>思<rt>おも</rt></ruby>う（我认为…）

表达观点。句子普通形 + と思う。

<ruby>明日<rt>あした</rt></ruby>は<ruby>雨<rt>あめ</rt></ruby>が<ruby>降<rt>ふ</rt></ruby>る**と<ruby>思<rt>おも</rt></ruby>います**。
- 我觉得明天会下雨。

この<ruby>映画<rt>えいが</rt></ruby>は<ruby>面白<rt>おもしろ</rt></ruby>い**と<ruby>思<rt>おも</rt></ruby>います**。
- 我觉得这部电影很有趣。
