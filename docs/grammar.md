# 日语基础语法

日语的语序通常是「主语 - 宾语 - 动词」(SOV)，而中文和英语是「主 - 动 - 宾」(SVO)。

## 基本句子结构

### 1. 肯定句

> **主题** + **は** + **对象** + **です**

- <ruby>私<rt>わたし</rt></ruby>は<ruby>学生<rt>がくせい</rt></ruby>です。
  - 我是学生。
- これはペンです。
  - 这是一支笔。

### 2. 否定句

> **主题** + **は** + **对象** + **ではありません** (或 じゃありません)

- <ruby>私<rt>わたし</rt></ruby>は<ruby>学生<rt>がくせい</rt></ruby>ではありません。
  - 我不是学生。

### 3. 疑问句

在句尾加上 **か**。

- <ruby>貴方<rt>あなた</rt></ruby>は<ruby>学生<rt>がくせい</rt></ruby>ですか？
  - 你是学生吗？

## 形容词

### い形容词

以 `い` 结尾。

- <ruby>美味<rt>おい</rt></ruby>しい - 好吃
- このリンゴは<ruby>美味<rt>おい</rt></ruby>しいです。
  - 这个苹果很好吃。
- <ruby>高<rt>たか</rt></ruby>い - 贵 / 高
- <ruby>新<rt>あたら</rt></ruby>しい - 新

**变形规则**：

| 时态 | 肯定 | 否定 (くない) |
|---|---|---|
| 非过去 | 美味しい | 美味**くない** / 美味しくありません |
| 过去 | 美味**かった** | 美味**くなかった** / 美味しくありませんでした |

### な形容词

词尾通常不是 `い`。修饰名词时加 `な`。

- <ruby>綺麗<rt>きれい</rt></ruby> - 漂亮/干净 (注意：虽以 `i` 结尾但属な形容词)
- <ruby>有名<rt>ゆうめい</rt></ruby> - 有名
- <ruby>彼女<rt>かのじょ</rt></ruby>は<ruby>綺麗<rt>きれい</rt></ruby>です。
  - 她很漂亮。
- <ruby>静<rt>しず</rt></ruby>かな<ruby>部屋<rt>へや</rt></ruby>。 (修饰名词)
  - 安静的房间。

**变形规则**：

| 时态 | 肯定 (です) | 否定 (ではない/じゃありません) |
|---|---|---|
| 非过去 | 綺麗です | 綺麗**ではありません** / 綺麗じゃありません |
| 过去 | 綺麗**でした** | 綺麗**ではありませんでした** / 綺麗じゃありませんでした |

## 存在句

表示人或物的存在。

### 1. あります - 无生命

用于物体、植物等。

> **场所** + **に** + **物体** + **が** + **あります**

- <ruby>机<rt>つくえ</rt></ruby>の<ruby>上<rt>うえ</rt></ruby>に<ruby>本<rt>ほん</rt></ruby>があります。
  - 桌子上有书。
- <ruby>公園<rt>こうえん</rt></ruby>に<ruby>木<rt>き</rt></ruby>があります。
  - 公园里有树。

### 2. います - 有生命

用于人、动物等。

> **场所** + **に** + **人/动物** + **が** + **います**

- <ruby>部屋<rt>へや</rt></ruby>に<ruby>猫<rt>ねこ</rt></ruby>がいます。
  - 房间里有猫。
- <ruby>教室<rt>きょうしつ</rt></ruby>に<ruby>学生<rt>がくせい</rt></ruby>がいます。
  - 教室里有学生。
