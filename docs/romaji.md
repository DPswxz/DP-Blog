# 罗马音 (Romaji)

罗马音是用拉丁字母标注日语读音的方法，常见于电脑输入和初学者学习。

## 输入法对照表

### 清音

| 平假名 | 罗马音 |
|---|---|
| あ い う え お | a i u e o |
| か き く け こ | ka ki ku ke ko |
| さ し す せ そ | sa si/shi su se so |
| た ち つ て と | ta ti/chi tu/tsu te to |
| な に ぬ ね の | na ni nu ne no |
| は ひ ふ へ ほ | ha hi hu/fu he ho |

### 浊音 (Dakuten)

| 平假名 | 罗马音 |
|---|---|
| が ぎ ぐ げ ご | ga gi gu ge go |
| ざ じ ず ぜ ぞ | za zi/ji zu ze zo |
| だ ぢ づ で ど | da di/ji du/zu de do |
| ば び ぶ べ ぼ | ba bi bu be bo |

### 半浊音 (Handakuten)

| 平假名 | 罗马音 |
|---|---|
| ぱ ぴ ぷ ぺ ぽ | pa pi pu pe po |

### 促音 (Sokuon)

小写的 `tsu` (っ) 表示停顿一拍，在罗马音中通常双写后续辅音。

- <ruby>学校<rt>がっこう</rt></ruby> (Gakkou) -> Gakkou
- <ruby>切手<rt>きって</rt></ruby> (Kitte) -> Kitte
- <ruby>雑誌<rt>ざっし</rt></ruby> (Zasshi) -> Zasshi
- <ruby>一杯<rt>いっぱい</rt></ruby> (Ippai) -> Ippai

### 长音 (Chouon)

元音发两拍，罗马音通常表示为元音重复或加 macron (ō/ū)，但输入时通常直接打出对应的元音。

- **aa**: <ruby>お母さん<rt>おかあさん</rt></ruby> (Okaasan)
- **ii**: <ruby>お兄さん<rt>おにいさん</rt></ruby> (Oniisan)
- **uu**: <ruby>数学<rt>すうがく</rt></ruby> (Suugaku)
- **ee**: <ruby>お姉さん<rt>おねえさん</rt></ruby> (Oneesan)
- **ei**: <ruby>先生<rt>せんせい</rt></ruby> (Sensei) -> 发音倾向于 "Sensee"
- **oo**: <ruby>大きい<rt>おおきい</rt></ruby> (Ookii)
- **ou**: <ruby>お父さん<rt>おとうさん</rt></ruby> (Otousan) -> 发音倾向于 "Otoosan"
