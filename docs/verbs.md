# 动词语气

动词的变形是日语语法核心之一。以下是三大类动词的基本分类。

## 动词分类

1. **一类动词**: 词尾是 <ruby>う<rt>u</rt></ruby> 段假名。
   - <ruby>行<rt>い</rt></ruby>く - 去
   - <ruby>飲<rt>の</rt></ruby>む - 喝
   - <ruby>死<rt>し</rt></ruby>ぬ - 死
   - <ruby>待<rt>ま</rt></ruby>つ - 等
   - <ruby>買<rt>か</rt></ruby>う - 买
   - <ruby>遊<rt>あそ</rt></ruby>ぶ - 玩
   - <ruby>読<rt>よ</rt></ruby>む - 读
   - <ruby>書<rt>か</rt></ruby>く - 写
   - <ruby>泳<rt>およ</rt></ruby>ぐ - 游泳
   - <ruby>話<rt>はな</rt></ruby>す - 说

2. **二类动词**: 词尾是 る 且倒数第二个假名是 い/え 段。
   - <ruby>食<rt>た</rt></ruby>べる - 吃
   - <ruby>見<rt>み</rt></ruby>る - 看
   - <ruby>起<rt>お</rt></ruby>きる - 起床
   - <ruby>寝<rt>ね</rt></ruby>る - 睡觉

3. **三类动词**: 不规则动词，只有两个。
   - する - 做
   - <ruby>来<rt>く</rt></ruby>る - 来

## 常见时态 (ます形 / 礼貌体)

将动词变成礼貌形 (<ruby>丁寧形<rt>ていねいけい</rt></ruby>)。

### 1. 肯定

- **一类动词**: 词尾う段 -> い段 + ます
  - <ruby>行<rt>い</rt></ruby>く -> <ruby>行<rt>い</rt></ruby>きます
  - <ruby>飲<rt>の</rt></ruby>む -> <ruby>飲<rt>の</rt></ruby>みます
  - <ruby>待<rt>ま</rt></ruby>つ -> <ruby>待<rt>ま</rt></ruby>ちます
  - <ruby>買<rt>か</rt></ruby>う -> <ruby>買<rt>か</rt></ruby>います

- **二类动词**: 去掉 る -> + ます
  - <ruby>食<rt>た</rt></ruby>べる -> <ruby>食<rt>た</rt></ruby>べます
  - <ruby>見<rt>み</rt></ruby>る -> <ruby>見<rt>み</rt></ruby>ます

- **三类动词**:
  - する -> します
  - <ruby>来<rt>く</rt></ruby>る -> <ruby>来<rt>き</rt></ruby>ます

### 2. 否定

将 ます 换成 ません。

- <ruby>食<rt>た</rt></ruby>べません (不吃)
- <ruby>行<rt>い</rt></ruby>きません (不去)

### 3. 过去式

- 肯定: ます -> ました
  - <ruby>食<rt>た</rt></ruby>べました (吃了)
- 否定: ません -> ません でした
  - <ruby>食<rt>た</rt></ruby>べません でした (没吃)

## て形 (连接形)

用于连接句子、请求 (～てください)、进行时 (～ている) 等。

### 一类动词

| 词尾 | 变化规则 | 示例 (辞书形 -> て形) |
|---|---|---|
| う, つ, る | 去掉 -> って | 買**う** -> 買**って** <br> 待**つ** -> 待**って** <br> 帰**る** -> 帰**って** |
| ぬ, ぶ, む | 去掉 -> んで | 死**ぬ** -> 死**んで** <br> 遊**ぶ** -> 遊**んで** <br> 読**む** -> 読**んで** |
| く | 去掉 -> いて | 書**く** -> 書**いて** <br> *特例: 行く -> 行って* |
| ぐ | 去掉 -> いで | 泳**ぐ** -> 泳**いで** |
| す | 去掉 -> して | 話**す** -> 話**して** |

### 二类动词

- 去掉 る -> + て
  - <ruby>食<rt>た</rt></ruby>べる -> <ruby>食<rt>た</rt></ruby>べて
  - <ruby>見<rt>み</rt></ruby>る -> <ruby>見<rt>み</rt></ruby>て

### 三类动词

- する -> して
  - <ruby>勉強<rt>べんきょう</rt></ruby>する -> <ruby>勉強<rt>べんきょう</rt></ruby>して
- <ruby>来<rt>く</rt></ruby>る -> <ruby>来<rt>き</rt></ruby>て

## 普通形

用于非正式场合 (朋友家人之间) 或书面语。

### 1. 辞书形 - 现在的肯定

动词的原形。

- <ruby>明日<rt>あした</rt></ruby>、<ruby>行<rt>い</rt></ruby>く。 (明天去。)
- 寿司を食べる。 (吃寿司。)

### 2. ない形 - 现在的否定

- **一类动词**: 词尾う段 -> あ段 + ない
  - <ruby>行<rt>い</rt></ruby>く -> <ruby>行<rt>い</rt></ruby>か**ない**
  - <ruby>読<rt>よ</rt></ruby>む -> <ruby>読<rt>よ</rt></ruby>ま**ない**
  - *注意: う -> わ + ない* (買**う** -> 買**わ**ない)

- **二类动词**: 去掉 る -> + ない
  - <ruby>食<rt>た</rt></ruby>べる -> <ruby>食<rt>た</rt></ruby>べない

- **三类动词**:
  - する -> しない
  - <ruby>来<rt>く</rt></ruby>る -> <ruby>来<rt>こ</rt></ruby>ない - *注意读音变化*

### 3. た形 - 过去的肯定

变化规则同て形 (只是把 て 换成 た, で 换成 だ)。

- **一类动词**:
  - 買**って** -> 買**った** (买了)
  - 読**んで** -> 読**んだ** (读了)
- **二类动词**:
  - <ruby>食<rt>た</rt></ruby>べ**て** -> <ruby>食<rt>た</rt></ruby>べ**た** (吃了)
- **三类动词**:
  - して -> した
  - きて -> きた

### 4. 过去否定形

先变成ない形，去掉 い -> 加かった。

- <ruby>行<rt>い</rt></ruby>かない -> <ruby>行<rt>い</rt></ruby>かな**かった** (没去)
- <ruby>食<rt>た</rt></ruby>べない -> <ruby>食<rt>た</rt></ruby>べな**かった** (没吃)
